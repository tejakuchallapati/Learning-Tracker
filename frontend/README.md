# Frontend

React + Vite app for Learning Tracker.

## Structure

- `src/app/` — entry point and routing
- `src/pages/` — route screens
- `src/components/` — UI grouped by feature (`layout`, `goals`, `dashboard`, `auth`, `landing`, `ui`)
- `src/services/api.js` — Axios client (set `VITE_API_BASE_URL` in `.env`)

## Commands

```bash
npm install
npm run dev      # http://localhost:3000
npm run build
```
